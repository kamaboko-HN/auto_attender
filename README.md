# What is this?
大江戸アナログ幼稚園の自動出席プログラムです。UNIVERSAL PASSPORT系の大学ポータルサイトの出席システムならだいたい使えるはず(要検証)
# How to use?
1.exeファイルを起動すると最初に「log.txt」「class.json」「login.json」が生成されます。
2.「login.json」に学籍番号とパスワードを「class.json」に曜日と時限ごとになにかしらの文字列を書いておいてください。(zoomURLとパスワードなど)
3.exeのショートカットをスタートアップに設定(ggrks)してPC起動時に立ち上げにする。

## ex:
- 授業やログイン情報を更新した際、アプリを立ち上げなおさずともコンソールに"update"と入れれば更新されます。
